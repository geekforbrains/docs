## Install

```sh
pip install operator-ai
```

Or install from source:

```sh
git clone https://github.com/geekforbrains/operator.git
cd operator
pip install -e .
```

## Initialize

```sh
operator init
```

This scaffolds `~/.operator/` with everything you need:
- `operator.yaml` — runtime config
- `SYSTEM.md` — system preamble
- `agents/operator/AGENT.md` — your first agent
- `skills/` — bundled starter skills

The command is idempotent — running it again won't overwrite your existing files.

## Configure

Edit `~/.operator/operator.yaml` with your model and transport:

```yaml
defaults:
  models:
    - "anthropic/claude-sonnet-4-6"
  max_iterations: 25
  env_file: "~/.env"

agents:
  operator:
    transport:
      type: slack
      bot_token_env: SLACK_BOT_TOKEN
      app_token_env: SLACK_APP_TOKEN
```

Set your API keys in `~/.env`:

```sh
ANTHROPIC_API_KEY=sk-ant-...
SLACK_BOT_TOKEN=xoxb-...
SLACK_APP_TOKEN=xapp-...
```

## Add yourself

Register as an admin so Operator knows who you are:

```sh
operator user add yourname --role admin slack YOUR_SLACK_USER_ID
```

## Run it

```sh
operator
```

That's it. Message your agent in Slack. 🐒

Logs go to `~/.operator/logs/operator.log`.

## Run as a service

Want it running in the background?

```sh
operator service install   # create launchd/systemd service
operator service start     # start it
```

See the [Service CLI](/cli/service) for more details.
