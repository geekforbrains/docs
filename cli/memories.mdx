---
title: "Removed: `operator memories`"
description: "Compatibility note for an older CLI page"
---

`operator memories` is not part of the current CLI.

Use the `operator memory` group instead:

| Old reference | Current command |
| --- | --- |
| `operator memories` | `operator memory list global` |
| `operator memories --scope ...` | `operator memory list <scope>` |
| `operator memories stats` | inspect scopes with `operator memory list ...` and `operator memory search ...` |

See [Memory CLI](/cli/memory) for the current command surface.
